Actualizar base de datos

Subí los nuevos archivos Excel de carteras. El sistema toma la fecha del nombre del archivo (YYYYMMDD_...). Podés reemplazar el período más viejo o agregar uno nuevo.

ℹ️ Los Excel deben tener la columna Sociedad Gerente completada antes de procesarlos con el script Python.
📂
Hacé clic o arrastrá archivos Excel (.xlsx) acá
Cada archivo = un viernes
Pasos para actualizar: 1. Agregá la columna Sociedad Gerente al nuevo Excel
2. Ejecutá: python procesar_carteras.py archivo_nuevo.xlsx archivo2.xlsx archivo3.xlsx
3. Reemplazá el data.js generado en la carpeta del sitio
4. Hacé push a GitHub → se actualiza automáticamente
